## java-exceptions

Repositório dedicado ao estudo de tratamento de exceções 
em Java, com implementações práticas de blocos try-catch-finally 
e criação de exceções personalizadas.

## Conceitos praticados

- Blocos try-catch-finally
- Exceções verificadas e não verificadas
- Criação de exceções personalizadas
- Hierarquia de exceções em Java

## Como rodar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe de cada exemplo
